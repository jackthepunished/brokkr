# Code of Conduct

This project follows the [Contributor Covenant v2.1][covenant].

In short: be respectful, assume good faith, give precise technical feedback,
and keep discussions focused on the work. Harassment of any kind is not
tolerated.

Report concerns to the project owner listed in `docs/plan.md`.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
